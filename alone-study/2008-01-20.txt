2. 리눅스 파일 시스템의 종류와 설명
-  mimix : 과거 minix에서 사용되어졌던 파일 시스템으로 가장 오래되고 기본이 되는 파일 시스템
- msdos : MS-DOS의 FAT 파일시스템과 호환을 지원하는 파일 시스템
- umsdos : MS-DOS 파일 시스템을 리눅스상에서도 긴 파일과 소유자, 접근허가, 링크와 장치 파일등을 사용할 수 있도록 확장한 시스템
- hpfs OS/2 : 읽기만 가능한 파일 시스템

3. du명령어 옵션
- -a : (all) 전부를 체크
- -k : KB (출력되는 기본 단위)
- -l : link (용량을 모두 계산)
- -c : 모든파일시스템 용량 합계
- -s : 사용량의 총합
- -S : 지정된 디렉토리 내의 파일 확인

4. 프로세스
