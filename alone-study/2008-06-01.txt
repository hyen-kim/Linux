< 1과목 >
 1. 리눅스 파일의 종류
- 특수파일
- 디렉토리파일
- 일반파일

2. 파일시스템의 사용한 용량과 사용 가능한 디스크 용량 정보를 보여주는 명령어
- df 

3. 리눅스에서 파티션을 생성하는 명령어 
- fdisk

4. 파일이나 디렉토리의 사용량을 볼 수 있는 명령어 
- du

5. 링크파일
- 하드링크는 원본파일과 inode값이 같다.
- 심볼릭링크는 디렉토리도 링크가 가능하다.
- 심볼릭링크는 원본파일이 삭제되면 사용할 수 없다.

6. 쉘의 특징
- 사용자가 입력한 명령을 해석하고 실행시키는 역할을 한다.
- 쉘 프로그램을 작성하여 복잡한 명령을 간단하게 수행한다.
- 사용자의 편의에 맞는 쉘로 변경이 가능하다.

7. Bourn Again Shell 이라고 불리우며, GNU 프로젝트의 일환으로 작성된 쉘
- bash

8. bash의 특징
- [Tab]키를 이용한 명령어 완성기능
- 명령어 history 기능
- 명령어 alias 기능

9. echo $SHELL 은 현재 쉘을 확인할 수 있다.

10. BASH에서 사용되는 특수문자
- ? : 단일 문자와 일치하는 파일 치환 문자
- * : 0개 이상의 문자와 일치하는 파일 치환 문자
- > : 표준출력을 파일에 기록하는 출력 리디렉션
- >> : 출력 리다이렉션 기호, 추가의 의미

11. 사용자가 로그아웃하거나 터미널창을 닫아도 해당 프로세스를 백그라운드로
전환하여 계속 작업될 수 있도록 해주는 명령
- nohup : 터미널을 닫아도 백그라운드로 작업

*) 추가
- fg : 포그라운드로 전환한다
- bg : 백그라운드로 전환한다

12. vi 편집기 입력모드 전환 방법
- I : 입력모드로 전환, 현재 줄의 앞에 삽입
- a : 입력모드로 전환, 커서 위치 뒤에서 삽입
- O : 입력모드로 전환, 커서가 있는 위에 삽입
- A : 입력모드로 전환, 현재 줄의 끝에 삽입
- o : 현재 줄 다음의 줄에 입력

데이터 블록을 의미하며, 패킷간 상호간섭을 최소화 하면서 망을 경유 하여 이동할 수 있다.

- 서로 다른 구조의 통신망을 연결할 때
